# 🦈 아구아구 (AguAgu) — Web Port

냥냥냥 캐주얼게임 **아구아구(eating)** 의 Android 원작을 **Phaser 3 + Vite** 로 포팅한 웹 버전입니다.

> 원작 영상: https://www.youtube.com/watch?v=4gDJozFp-i4

---

## 🚀 개발 서버 실행

```bash
npm install
npm run dev
# → http://localhost:5173
```

빌드:
```bash
npm run build
```

## 📖 게임 규칙

오른쪽에서 컨베이어 벨트를 타고 물체들이 왼쪽으로 이동합니다.
**두 개의 도어**를 열고 닫아 아이템을 올바른 곳으로 보내세요!

```
[스폰] →→→→ [도어2 닫힘] →→→→ [도어1 닫힘] →→→→ [상어]
              (오른쪽 도어)       (왼쪽 도어)
```

### 아이템 처리 방법

아이템은 오른쪽에서 왼쪽으로 이동합니다. **오른쪽 도어 → 왼쪽 도어 → 상어** 순서로 통과합니다.

| 아이템 | 왼쪽 도어 (보석 바구니) | 오른쪽 도어 (쓰레기통) | 상어에게 도달 |
|---|---|---|---|
| 🐟 **물고기** | ❌ `-100점`, HP -1 | ❌ `-100점`, HP -1 | ✅ `+100점`, HP +1 (2마리마다) |
| 💎 **루비** | ✅ `+70점` | ❌ `-70점`, HP -1 | ❌ `-150점`, HP -1 |
| 🗑️ **쓰레기** | ❌ `-70점`, HP -1 | ✅ `+70점` | ❌ `-150점`, HP -1 |

> 💡 **요약**: 물고기는 두 도어를 **모두 닫아** 상어까지 보내고, 루비는 **왼쪽 도어** 열기, 쓰레기는 **오른쪽 도어** 열기

### HP 시스템
- HP는 **2초마다 자동 감소** (최대 12)
- 물고기 2마리 연속 먹이면 HP +1 회복
- HP가 0이 되면 게임 오버

## 📈 난이도 (점수 기준)

| 점수 구간 | 아이템 종류 | 스폰 간격 |
|---|---|---|
| 0 ~ 1,000 | fish01, ruby, trash01 | 1.0초 |
| 1,001 ~ 3,500 | fish01, ruby, trash01 | 0.8초 |
| 3,501 ~ 6,000 | + trash02 추가 | 0.8초 |
| 6,001 ~ 10,000 | + trash02 | 0.6초 |
| 10,001 ~ 15,000 | + fish02 추가 | 0.6초 |
| 15,001 ~ 20,000 | + trash03 추가 | 0.6초 |
| 20,001 ~ | + fish03 추가 | 0.6초 |

## 🗂️ 프로젝트 구조

```
aguagu-web/
├── src/
│   ├── main.js               # 게임 엔트리 (GAME_WIDTH=1280, GAME_HEIGHT=720)
│   ├── scenes/
│   │   ├── BootScene.js      # 에셋 프리로더
│   │   ├── StartScene.js     # 타이틀 화면
│   │   ├── GameScene.js      # 메인 게임플레이
│   │   └── ShopScene.js      # 상점 화면
│   └── config/
│       └── GameConfig.js     # 상수 (HP, 난이도, 점수)

├── index.html
├── package.json
└── vite.config.js
```
